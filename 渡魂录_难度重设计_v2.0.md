# 渡魂录 SOUL FERRY — 难度与成瘾性重设计文档

> 基于 design_numbers.md v1.0 的深度评审，识别核心问题并给出完整改版数值与机制。
> 核心判断：**三个系统各自独立运转，没有相互咬合。难度来自数值，而非决策压力。**

---

## 一、问题诊断

### 渡化系统为什么没有成瘾性

| 现象 | 根本原因 |
|------|---------|
| 攒够情绪值就触发，像填表 | 条件是静态的，不随战况变化 |
| 渡化过程中敌人继续正常攻击 | 没有「渡化窗口」的风险感 |
| 奖励只有 +3 MaxHP | 收益太小，不值得专门追求 |
| 镇压更快、更稳 | 两条路线没有真实博弈 |
| 对话触发后战斗基本结束 | 戏剧张力全在对话，不在战斗 |

### 战斗系统为什么不够成瘾

| 现象 | 根本原因 |
|------|---------|
| 情绪失调惩罚太轻 | 喜失调只+20%伤，惧失调只弃1张 |
| 敌人情绪施压太弱 | 最高只有+2，很少能真正推到失调 |
| 定情绪无失调 | 没有风险的情绪不需要管理 |
| Boss 二阶段只是倍率变化 | 没有质变，只是更硬的同一个敌人 |
| 休息回血30%太好算 | 无需权衡就选回血，无决策成本 |
| 锁链每回合-1衰减 | 施锁的「存在价值」被自然侵蚀 |
| 普通敌人 HP 30-55 | 几张牌就清场，没有学习怪物行为的必要 |

## 二、情绪系统重设计

### 2.1 情绪上限从 5 改为 6，失调阈值保持 ≥4

**理由**：5上限+失调阈值4，玩家只有1格「安全余量」。改为6上限后：
- 4 = 失调触发（已有惩罚）
- 5 = 深度失调（新增惩罚，更危险）
- 6 = 崩溃状态（新增，严重惩罚）

```
const MAX_VALUE = 6        # 原5
const DISORDER_THRESHOLD = 4
const DEEP_DISORDER = 5    # 新增
const COLLAPSE = 6         # 新增，触发崩溃
```

### 2.2 失调分级惩罚（全面加重）

| 情绪 | 失调（≥4） | 深度失调（≥5）| 崩溃（=6，封顶触发） |
|------|-----------|-------------|------------------|
| 怒 | 无法打定系/护盾牌 | 无法打定系/护盾牌 + 每回合开始自动攻击自身3点 | 本回合所有手牌变为「愤怒」牌（只能打出，不能选择） |
| 惧 | 每回合随机弃1张牌 | 随机弃2张 + 无法打0费牌 | 本回合跳过（无法行动，直接进入敌人回合） |
| 悲 | 无法打治疗牌 | 无法打治疗牌 + 手牌上限-2 | 将手中所有治疗牌替换为「泪伤」（悲系普通攻击），无法移除 |
| 喜 | 敌人伤害×1.2 | 敌人伤害×1.4 + 无法使用遗物主动效果 | 敌人本回合无视护盾直接扣HP |
| 定 | **无效果（保留现状）** | 定≥5：每回合结束情绪全部+1（定越高越失控） | 定=6崩溃：五情全部强制变为3（一次性重置，但下回合开始全部反弹+2） |

> **定情绪的改动是整个系统最关键的一处**。定情绪过去没有失调，等于无风险的纯收益。深度失调（≥5）时反而会推升其他情绪，让「堆满定」变成一把双刃剑。

### 2.3 主导情绪增幅调整（加入负面意味）

主导情绪不只有加成，还有代价。**每种情绪的主导状态同时给与一个「主导弱点」**：

| 主导情绪 | 原加成（保留） | 新增主导弱点 |
|---------|------------|------------|
| 怒 | 攻击×1.3 | 受到「定系攻击」时伤害×1.5 |
| 惧 | 额外摸牌+2 | 无法主动弃牌（弃牌机制被封锁） |
| 悲 | 吸血×1.5 | 回合结束时定情绪-1（定被悲侵蚀） |
| 喜 | 回复×1.5 | 定系牌费用+1（喜时无法低费防御） |
| 定为主导时 | 费用-1（保留） | 无弱点（但定成为主导意味着其他情绪都偏低，系统自平衡） |

### 2.4 情绪自然衰减规则改动

**当前**：非主导、非定情绪每回合-1。

**改动**：
- 非主导情绪每回合 **-1**（不变）
- **主导情绪每2回合 -1**（不再永远不衰减，形成「主导情绪也会消退」的紧迫感）
- **定情绪每3回合 -1**（定情绪最稳定，但不是永恒的）
- 若某情绪处于失调（≥4），该情绪衰减速率变为每回合 **-2**（失调状态下身体在快速宣泄）

## 三、渡化系统重设计

> 核心重设：**渡化不是攒情绪然后触发，而是在敌人的「情绪涌动」窗口内完成共鸣。错过窗口、或在错误时机触发，渡化会崩溃并产生惩罚。**

### 3.1 渡化窗口机制（核心新增）

每个敌人的情绪施压行动，现在具有双重功能：**同时压迫玩家情绪，同时开放「渡化频率」**。

```
渡化频率（Du_Hua_Frequency）：
  - 每个敌人有一个隐藏的「频率值」，0-100
  - 敌人每次使用「情绪施压」行动时，频率+20
  - 频率≥60时，进入「渡化窗口」状态（UI显示：敌人形态微微透明，出现金色波纹）
  - 频率每回合自然衰减 -15
  - 频率<30时，进入「闭合」状态（渡化不可触发，即使情绪条件满足）
```

**这意味着**：敌人越主动施压玩家情绪，越容易被渡化；但敌人施压越多，玩家情绪风险越高。这是核心博弈。

### 3.2 渡化条件改为「情绪共振」而非「情绪阈值」

**当前**：悲≥3 AND 惧≥2 → 触发渡化（静态阈值）

**改为**：
```
渡化条件检测（每次玩家打出牌后检测一次）：

  1. 渡化窗口开放中（频率≥60）
  2. 玩家的「主导情绪」与敌人当前「主导情绪」相匹配（±1以内）
  3. 本回合已打出至少1张与该情绪对应的牌
  
→ 三条同时满足：进入渡化判定
```

**为什么更好**：
- 玩家需要「读取敌人的情绪状态」，而不只是等自己情绪积累
- 敌人主导情绪会随战斗变化（见3.3），创造动态的瞄准目标
- 与其他情绪互动更丰富（你的悲很高但敌人现在主导怒，不匹配）

### 3.3 敌人情绪系统（新增）

**敌人现在也有情绪值**，随自身行动和玩家行动变化：

```
敌人情绪规则：
  - 敌人每次使用「攻击类」行动：怒+1
  - 敌人每次使用「防御类」行动：定+1
  - 敌人每次使用「情绪施压」行动：对应情绪+2
  - 玩家对敌人造成伤害时：敌人惧+1（每回合最多触发2次）
  - 敌人情绪也有主导判定（4种非定情绪的最高值）
  
  敌人情绪≥4（失调）时：
    - 不是惩罚，而是「执念暴走」：该行动的效果×1.5，但下回合该情绪-3
    - 设计意图：失调的敌人更危险但也更脆弱，是攻击或渡化的好时机
```

**敌人情绪对渡化的影响**：
- 敌人情绪越接近失调，渡化频率积累越快（执念强的时候更容易被渡化）
- 玩家使用与敌人主导情绪相同的牌时，渡化频率额外+10

### 3.4 渡化进度改为「紧绷的三阶段」

**当前**：渡化进度是简单的百分比累积。

**改为**：
```
渡化阶段（Du_Hua_Stage）：0 / 1 / 2 / 完成

  阶段0→1：需要在「渡化窗口开放」且「情绪共振」时打出3张对应牌
            → 第一阶段触发：敌人「愣住」（跳过下1次行动），触发第一段对话
            
  阶段1→2：需要在第一阶段完成后的3回合内再次完成情绪共振（时间窗口）
            → 若3回合内未完成：渡化「中断」，退回阶段0，敌人获得「执念爆发」Buff（攻击×1.5持续2回合）
            → 若完成：触发第二段对话
            
  阶段2→完成：需要HP≥50%（情绪必须稳定，自身不能太狼狈）
              → 若HP<50%：渡化完成受阻，提示「你自己也快撑不住了，对方感受不到你的真心」
              → 完成渡化：触发结局
```

### 3.5 渡化失败的惩罚（让渡化有赌注）

渡化尝试失败（中断退回阶段0）时，引入惩罚：
- 第1次中断：敌人获得「执念」Buff，攻击+3，持续2回合
- 第2次中断：敌人HP回复15%（执念越强越难被打死），渡化频率自然增速减半
- 第3次中断：该场战斗渡化路线永久关闭（只能镇压），且玩家所有情绪+2（情绪污染）

> **设计意图**：渡化不再是「顺手完成」的额外奖励路线，而是需要专门准备且有风险的高收益路线。

### 3.6 渡化成功奖励重设计（让渡化值得追求）

**当前**：通过「渡魂册」遗物获得永久+3 MaxHP。

**改为分级奖励**：

| 渡化质量 | 条件 | 奖励 |
|---------|------|------|
| 初渡（勉强完成） | 完成渡化，HP<40%，尝试过失败 | +3 MaxHP（原奖励），随机1张普通牌 |
| 善渡（稳定完成） | 完成渡化，HP≥50%，无失败 | +5 MaxHP，随机1张稀有牌，金币+20 |
| 无悔渡（完美完成） | 完成渡化，HP≥70%，全程零失败，且与敌人情绪完美共振（全部3阶段主导情绪匹配） | +8 MaxHP，随机1张传说/史诗牌，金币+40，路线倾向计数×2 |

> 「无悔渡」是技术天花板，让老手有追求。

## 四、战斗数值全面修订

### 4.1 普通敌人重设计

**当前问题**：HP 30-55，攻击力低，情绪施压弱，可以无脑通过。

**改动原则**：普通敌人不再是用来练手的，而是**情绪教学机器**——每种敌人都强迫玩家学会处理一种情绪威胁。

| 名称 | 层级 | HP（原→新） | 攻击力（原→新） | 情绪施压（原→新） | 新增行为 |
|------|------|-----------|--------------|----------------|---------|
| 孤魂游灵 | 1 | 30→**45** | 8→**10** | 惧+1→**惧+2** | 新增：「游荡」（跳过行动但惧+1，每2回合一次） |
| 怨鬼 | 1 | 40→**60** | 10→**13** | 悲+1怒+1→**悲+2怒+1** | 新增：「怨气积聚」（蓄势行动，下次攻击×2） |
| 摄魅眼 | 2 | 45→**75** | 12→**16** | 惧+2→**惧+3** | 新增：「摄魂凝视」（跳过玩家下1次摸牌） |
| 缕头鬼 | 2 | 55→**85** | 14→**18** | 悲+1→**悲+2** | 新增：「拉扯」（玩家随机1张手牌的费用+2，持续1回合） |
| 闪火鬼 | 3 | 50→**90** | 14→**20** | 怒+2→**怒+3** | 新增：「烈火外泄」（玩家不打出攻击牌时，每回合怒+1） |

**新增第1层精英怪（第1层后期节点出现）**：

| 名称 | HP | 攻击 | 情绪施压 | 特殊能力 |
|------|-----|------|---------|---------|
| 执念老妪 | 110 | 15 | 悲+2/回合 | 「情绪感染」：玩家悲每达到2，对玩家施加「执念」（1回合情绪不衰减）；HP<40%时悲施压变为+3 |
| 幽火武士 | 100 | 22 | 怒+2/回合 | 「怒烈格挡」：玩家打出护盾牌时，武士怒+1（每攻击次数=己方护盾量，鼓励进攻路线对抗此敌）|

### 4.2 Boss 重设计（三个Boss全面强化）

#### Boss 1：水鬼·望归（江盼鱼）

**当前HP**：80/120，**新HP**：100/160

**Phase 1（HP 100-50%）**：

| 行动 | 数值（原→新） | 权重 | 备注 |
|------|------------|------|------|
| 水涌 | 12→**16** | 35% | 普通攻击 |
| 浪击（全体） | 7→**10** | 20% | AOE |
| 七年等候 | 悲+1→**悲+2** | 30% | 情绪施压（强化） |
| 水中沉潜 | 回血8→**12** | 15% | 自愈（强化） |

**Phase 2（HP 50%以下）—— 全面改变行为模式**：

新行为：「绝望潮」机制启动
- 每回合必定执行「涨潮」（不计入行动次数）：玩家悲+1，**渡化频率-10**（主动阻碍渡化）
- 若渡化频率被压低到0，该回合「渡化窗口永久关闭」（除非特定牌/遗物重置）

| 行动 | 数值 | 权重 |
|------|------|------|
| 七年长哭 | 伤害14 + 悲+3 | 40% |
| 召唤潮汐 | 连续攻8×3次 | 25% |
| 溺水记忆（新） | 玩家手牌最多的情绪对应牌全部变成「溺水」（无效果的占位牌，持续1回合） | 20% |
| 护盾+漂流（新） | 自护盾25 + 渡化频率-20 | 15% |

**渡化条件（原）**：悲≥3且惧≥2
**渡化条件（新）**：
- 在渡化窗口开放时：玩家「悲」为主导情绪 + 本回合打出1张悲系牌 + HP≥50%（代表「你撑得住，才能渡她」）
- 特殊：「忘川引」（渡化进度+2的牌）在Phase 2中可直接+30%渡化进度（象征性价值翻倍）

#### Boss 2：旱魃·焦骨（宋兰秀）

**当前HP**：100/150，**新HP**：130/190

**Phase 1（HP 130-65%）**：

| 行动 | 数值（原→新） | 权重 |
|------|------------|------|
| 旱怒 | 15→**20** | 30% |
| 烈火蔓延（全体） | 9→**12** | 20% |
| 无人相信 | 怒+1→**怒+2** | 25% |
| 六十年执念 | 自盾12→**16** | 25% |
| 火焰DOT | 5×3回合→**7×4回合** | 在特定条件下释放 |

**Phase 2（HP 65%以下）—— 「旱魃之怒」阶段**：

新机制：「旱野焦灼」全局Buff启动
- 全局环境效果：玩家每回合开始时，若手中有2张以上相同系的牌，其中1张随机受「灼烧」（费用+1，无法通过打出移除，1回合后自然消失）
- 意图：打乱玩家的稳定循环，强迫随机应对

| 行动 | 数值 | 权重 |
|------|------|------|
| 烈火审判（新） | 穿甲伤害25 + 玩家怒+3 | 35% |
| 焦骨蔓延（新） | 对玩家施加「焦骨」：2回合内每打出牌就受1点DOT | 25% |
| 旱地长叹（新） | 清除玩家本回合所有护盾，然后自回血10 | 20% |
| 执念加冕（新） | 自盾20 + 下2回合受到伤害-50%（打破这个护盾需要特定情绪牌或渡化推进） | 20% |

**渡化条件（原）**：定≥3选择「我相信你」
**渡化条件（新）**：
- 玩家「定」为主导情绪（代表内心稳定足以听她说话）
- **同时**玩家本场战斗怒情绪至少一次达到过≥3（代表玩家自己也感受过愤怒，能理解她的怒）
- 在渡化窗口：打出「慈悲感化」或「忘川引」

#### Boss 3：鬼新娘·素锦（沈素锦）

**当前HP**：120/180，**新HP**：150/220

**三阶段渡化（沈素锦的特殊机制全面强化）**：

素锦的战斗本身就是渡化过程，她的行动模式随「渡化阶段」改变：

| 渡化阶段 | 素锦状态 | 行动模式 |
|---------|---------|---------|
| 阶段0（未触发） | 「空」——无意识的索取 | 攻击14-18，情感真空（玩家情绪-1），索取喜意（玩家喜-1） |
| 阶段1（第一段对话后） | 「好奇」——开始感知 | 攻击降至10，增加「困惑凝视」（玩家喜系牌效果+20%，但她同时有20%概率取消攻击） |
| 阶段2（第二段对话后） | 「触动」——接近理解 | 停止所有攻击性行动，改为「情绪接收」（每回合玩家打出的牌效果对渡化进度有贡献） |
| 阶段3（渡化完成） | 结局 | — |

**渡化阶段1触发条件（新）**：
- 玩家连续2回合主导情绪为「喜」
- 且本回合打出「红尘一笑」或「华光散」（象征展示喜悦，而非在战斗）

**渡化阶段2触发条件（新）**：
- 阶段1完成后3回合内
- 玩家当前HP**低于**最大HP的60%（你自己也不完美，才能真正理解空缺）
- 打出「渡引咒」或「喜泉」

**渡化阶段3触发条件（新）**：
- 阶段2完成后
- 玩家说出「我喜欢今天」（对话选项，而非情绪条件——她需要的是一句真话）

**Phase 2 触发（HP 40%以下，只在渡化失败/中断时触发）**：

| 行动 | 数值 | 权重 |
|------|------|------|
| 狂暴连击 | 5+手牌数×2 | 50% |
| 情感真空（强化版） | 所有情绪-2 | 30% |
| 彻底崩溃（新） | 对玩家造成25点穿甲伤害 + 玩家所有喜系牌本场失效 | 20% |

### 4.3 全局战斗数值调整

**玩家护盾规则调整**：
- 护盾不再默认回合结束清零改为：**护盾每回合末减半（向下取整）**，不是清零
- 例：上回合护盾15 → 本回合开始时变为7（而不是0）
- 效果：护盾管理更有持续价值，不用每回合从零开始，但也不是永久存在

**费用与摸牌调整**：
- 第一回合摸牌：4张（不变）
- 后续每回合摸牌：**3张**（从当前的3张保持，但）加入「情绪奖励摸牌」：
  - 回合开始时，若玩家处于任意情绪主导状态：额外摸1张（从0变为稳定的奖励）
  - 若玩家同时有2种情绪≥2：额外再摸1张（鼓励多元情绪策略）

**休息点选项调整**：
- 回复HP：**20%**（从30%降低，让HP管理更紧张）
- 升级牌卡：保持不变
- 移除牌卡：**免费**（从75金降为0金）
- **新增第4选项**：「焠炼」——消耗25 HP，将一张牌升级后再锻造一次（相当于强化+锻造同时完成，代价是HP）

## 五、敌人行为复杂度升级

### 5.1 意图系统新增「复合意图」

当前：敌人每回合执行1个行动。

**改为**：敌人行动现在有「主意图」和「副意图」，同时显示两个：

```
意图组合示例（怨鬼）：
  主意图：攻击13（红色图标）
  副意图：怨气积聚（橙色图标，蓄势效果）
  
  → 玩家看到这一组意图时，需要同时考虑：「我能否在挨了13点伤害后，还处理好下回合的×2攻击？」
```

副意图规则：
- 副意图不可被格挡，不占行动次数
- 副意图固定在「主意图执行后」触发
- 敌人HP进入后50%时，副意图威力×1.5

### 5.2 「执念计数器」（新增普适机制）

每个敌人头顶新增**执念计数器**（0-5），代表亡魂对世界的「执念强度」：

```
执念计数器规则：
  - 每回合开始：+1（自然积累，亡魂的执念随时间加深）
  - 受到渡化类牌攻击：-1（渡化在削弱执念）
  - 受到纯攻击伤害：+0（镇压不影响执念）
  - 渡化窗口开放需要：执念计数器≥3（执念够深才有被渡化的可能）
  - 执念计数器=5：进入「执念爆发」（本回合行动效果×2，然后-3）
  
UI显示：敌人头顶的五个小格，满格时橙色发光
```

**这个机制的战略含义**：
- 快攻（镇压）路线：在执念爆发前把敌人击败，避开高执念的危险时刻
- 渡化路线：需要等执念积累到≥3，但也要在执念=5爆发前完成共振
- 二者形成真实的时间节奏博弈

### 5.3 精英怪「技能模式」（新增）

第2、3层的精英怪增加「技能模式」，会使用需要特定策略应对的能力：

| 技能名 | 效果 | 应对策略提示 |
|--------|------|------------|
| 情绪屏蔽 | 下回合玩家的指定情绪（随机）+0（增加无效） | 需要切换到该情绪的相邻情绪 |
| 印记抹除 | 清除玩家场上的所有印记（阮如月噩梦） | 需要在此行动前触发共鸣 |
| 锁链切断 | 清除所有锁链标记（沈铁钧噩梦） | 需要在此行动前引爆溅射 |
| 空度重置 | 将无名的空度强制变为5（中段，清除分段加成） | 需要提前储备空度调整牌 |
| 护盾生长 | 每回合自动+8护盾，叠加3回合（共+24） | 需要穿甲伤害或破盾组合 |

## 六、成瘾性机制设计

> 成瘾性不来自难度，而来自**「每次死亡都感觉我差一点就成功了」**的张力感。

### 6.1 「最后一口气」系统（新增）

当玩家HP≤10%时，进入「余光状态」：
- 持续2回合：每回合开始摸1张额外牌
- 2回合内若成功完成渡化：奖励翻倍（稀有→传说级）
- 2回合内若HP归零：**不立即失败**，而是「渡化失败结局」——玩家可以继续一段额外剧情，但无法获得本关奖励

**设计意图**：让「低血量」从「快失败了」变成「高风险高回报的机会窗口」。

### 6.2 「情绪遗留」系统（新增跨战斗机制）

战斗结束后，主导情绪不会归零，而是以「余韵」形式保留到下一场战斗：

```
情绪余韵规则：
  战斗结束时，若玩家主导情绪值≥3：
    → 下一场战斗开始时，该情绪起始为1（而不是0）
    → 同时获得「余韵加持」：与该情绪对应的牌，本场费用-1（第一回合有效）
  
  若通过渡化获胜：
    → 「渡化余韵」：下场战斗渡化频率初始值+20（更快进入渡化窗口）
    
  若通过镇压获胜（敌人HP=0）：
    → 「怒意余韵」：下场战斗攻击×1.1（第一回合有效）
```

**设计意图**：每场战斗的结局影响下一场的开局，形成「情绪剧情线」而不是每场都从零开始。

### 6.3 「命运颠倒」事件（新增高风险高回报事件节点）

地图上新增一类节点「命运颠倒」（每层1-2个，形状特殊）：

进入后必须在两个极端选项中选一个：

| 选项 | 结果 |
|------|------|
| A（高收益高风险） | 获得一件史诗遗物，但本层剩余所有战斗的敌人HP+30% |
| A（高收益高风险） | 获得永久+20 MaxHP，但移除牌组中1张最强的牌 |
| B（稳定） | 获得80金，下一场战斗的渡化条件简化（降低共振要求） |

**设计意图**：强制玩家做出带有风险代价的决策，而不只是选择「最优解」。

### 6.4 连击奖励系统（新增）

连续成功渡化（不中断、不失败）会累计「渡化连击」：

| 连击数 | 额外奖励 |
|--------|---------|
| 2连 | 下场战斗开始时敌人执念计数器初始为2（更快进入渡化窗口） |
| 3连 | 解锁「深度渡化」选项：可选择一张牌变为「渡化专属」版（效果质变） |
| 4连 | 获得一件「渡化传承」遗物（专属稀有遗物池） |
| 5连+ | 每次渡化奖励变为最高级（无悔渡品质），无论实际完成质量 |

若中途发生一次镇压：连击重置为0。

## 八、给开发的实现提示词

```
【任务】根据数值重设计文档，对渡魂录的核心难度和成瘾性系统进行全面改造。

【最高优先级改动（3个核心变更）】

变更1：EmotionManager 情绪分级失调
  将情绪最大值从5改为6，新增深度失调（≥5）和崩溃（=6）分支：
  
  func check_disorder(emotion_type: String, value: int) -> void:
      if value >= 6:
          _trigger_collapse(emotion_type)       # 崩溃：见表格
      elif value >= 5:
          _trigger_deep_disorder(emotion_type)  # 深度失调：见表格  
      elif value >= 4:
          _trigger_disorder(emotion_type)       # 原有失调：保持
  
  定情绪深度失调（≥5）的特殊效果：
  func _on_turn_end_calm_deep():
      # 定≥5时：其他四种情绪各+1（定越高越失控）
      for emotion in ["nu","ju","bei","xi"]:
          add_emotion(emotion, 1)

变更2：渡化窗口系统（DuHuaWindowSystem）
  新增 du_hua_frequency: int = 0 变量（每个敌人独立）
  
  触发规则：
    敌人使用「情绪施压」行动 → du_hua_frequency += 20
    每回合结束 → du_hua_frequency -= 15（自然衰减）
    du_hua_frequency >= 60 → 渡化窗口开放
  
  渡化条件由静态情绪阈值改为：
    1. du_hua_frequency >= 60
    2. 玩家主导情绪 == 敌人主导情绪（±1以内）
    3. 本回合已打出对应情绪的牌
  
  渡化中断惩罚（第2次中断时）：
    敌人回血15%，du_hua_frequency 自然积累速率减半

变更3：护盾改为「回合末减半」而非清零
  BattleStateMachine 在 STATE_TURN_END 阶段：
  # 原代码
  player_shield = 0
  
  # 改为
  player_shield = player_shield / 2  # 向下取整
  # 注意：铁身/静若磐石等「护盾不清零」的牌，效果改为「护盾不减半」

【次优先级改动（4个配合变更）】

变更4：执念计数器（EnemyObsessionCounter）
  每个敌人新增 obsession_count: int = 0（0-5）
  回合开始：obsession_count = min(obsession_count + 1, 5)
  玩家打渡化类牌：obsession_count -= 1
  obsession_count == 5：执念爆发，行动效果×2，然后 obsession_count -= 3

变更5：情绪余韵（跨战斗）
  GameState 新增：
  var lingering_emotion: String = ""   # 上场战斗主导情绪
  var lingering_value: int = 0          # 余韵初始值
  var lingering_type: String = ""       # "purification" / "suppression"
  
  战斗结束时：
  if current_dominant_emotion_value >= 3:
      GameState.lingering_emotion = dominant_emotion
      GameState.lingering_value = 1
  
  下场战斗开始时：
  if lingering_emotion != "":
      EmotionManager.add_emotion(lingering_emotion, lingering_value)
      # 对应情绪牌本场第一回合-1费（通过 card_cost_modifier 实现）

变更6：Boss 二阶段行为质变
  每个Boss的Phase 2需要完全替换行动池（不只是数值乘倍率）
  在 BattleStateMachine._enter_phase_two() 中：
  current_enemy_actions = enemy_data.get("phase_2_actions", [])
  # 确保 enemies.json 中有独立的 "phase_2_actions" 数组

变更7：渡化成功分级奖励
  在 BattleStateMachine._complete_purification() 中：
  var quality = _calculate_purification_quality()
  # quality: "minimal" / "stable" / "perfect"
  
  func _calculate_purification_quality() -> String:
      if purification_failures == 0 and current_hp >= max_hp * 0.7 and emotion_sync_perfect:
          return "perfect"
      elif purification_failures == 0 and current_hp >= max_hp * 0.5:
          return "stable"
      else:
          return "minimal"

【数值文件需要同步修改的 JSON 字段】

enemies.json（每个敌人新增）：
  "obsession_init": 0,              # 执念计数器初始值（boss可设为1）
  "dominant_emotion": "grief",       # 敌人当前主导情绪（可随行动改变）
  "phase_2_actions": [...],          # 完全替换的Phase 2行动池
  "purification_window_gain": 20    # 每次情绪施压增加的渡化频率量

配合以上改动，需要更新以下数值（见文档表格）：
  - 所有普通敌人HP和攻击力（按表格）
  - Boss 1/2/3 的HP和Phase 2行动（按表格）
  - EmotionManager.MAX_VALUE: 5 → 6
  - GameState.STARTING_HP：阮80→75，沈100→95，无名60→55
  - RestScene 回血比例：0.3 → 0.2
```

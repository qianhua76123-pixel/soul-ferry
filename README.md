# 渡魂录 SOUL FERRY

> **Roguelite Deckbuilder** · Godot 4.6 · Solo + AI 辅助开发

一款以中国民俗为底色的卡牌 Roguelite。玩家扮演渡魂人，使用五情印记与铁链锁链，在镇压与渡化两条路线之间做出选择，引导困于世间的亡魂走向解脱或被压制。

---

## 游戏概述

渡魂录是一款水墨剪纸风格的 Roguelite 卡牌游戏。玩家在阳间与幽冥交界处穿行，面对带着执念无法离去的亡魂。游戏提供三名可玩角色：庙祝阮如月（渡化偏向）、老捕快沈铁钧（镇压偏向）、无面人无名（均衡），以及渡化与镇压两条胜利路线。核心机制为五情情绪系统（怒/惧/悲/喜/定），情绪值积累可解锁战斗增益，但超过失控阈值将带来惩罚。每场战斗可以选择击败亡魂（镇压）或满足其情绪条件令其放下执念（渡化），两种路线奖励不同，共同构成叙事体验。

## 五情情绪系统

五情：**怒（rage）** / **惧（fear）** / **悲（grief）** / **喜（joy）** / **定（calm）**

- 情绪值范围：0–5，不可溢出
- 主导情绪：除定外，数值最高的情绪为主导，提供战斗加成
- 每回合结束：非主导且>0的情绪自动-1（定不衰减）

**主导情绪加成：**

| 情绪 | 主导加成 |
|------|---------|
| 怒 | 攻击伤害×1.3 |
| 惧 | 摸牌+2/回合 |
| 悲 | 吸取效果×1.5 |
| 喜 | 回复效果×1.5 |
| 定≥3 | 全牌费用-1 |

**失控阈值（≥4）及失控效果：**

| 情绪 | 失控效果 |
|------|---------|
| 怒≥4 | 无法打出定系牌和护盾牌 |
| 惧≥4 | 每回合开始随机弃1张牌 |
| 悲≥4 | 无法打出任何治疗牌 |
| 喜≥4 | 敌人伤害全场×1.2 |
| 定 | 无失调效果（专属共鸣无失控惩罚） |

**印记共鸣（默认阈值3层触发）：**

| 印记 | 共鸣效果 |
|------|---------|
| 悲共鸣 | 对敌人造成8点伤害 |
| 惧共鸣 | 跳过敌人下一次行动 |
| 怒共鸣 | 对敌人造成15点穿甲伤害 |
| 喜共鸣 | 回复8点HP |
| 定共鸣 | 本回合所有牌费用+1（获得1费） |
| 五情共鸣 | 五种印记同时存在→渡化进度+50%+20点穿甲伤害 |

## 运行方法

1. 安装 [Godot 4.6](https://godotengine.org/)
2. 克隆仓库：`git clone https://github.com/qianhua76123-pixel/soul-ferry.git`
3. 用 Godot 编辑器打开项目（`project.godot` 在根目录）
4. 运行主场景（F5 或点击播放按钮）

> ⚠️ 项目使用 GDScript 4 严格模式，需要 Godot **4.6** 及以上版本。

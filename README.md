# 渡魂录 SOUL FERRY

> **Roguelite Deckbuilder** · Godot 4.6 · Solo + AI 辅助开发

一款以中国民俗为底色的卡牌 Roguelite。玩家扮演渡魂人，使用五情印记与铁链锁链，在镇压与渡化两条路线之间做出选择，引导困于世间的亡魂走向解脱或被压制。

### 双路线系统

每场战斗有两种结局选择：

- **镇压路线**：纯战斗击败敌人，奖励牌卡与材料。适合不追求叙事的快速推进。
- **渡化路线**：理解亡魂的执念，使用特定情绪组合化解，奖励更丰厚，并推进主线叙事。

## 可玩角色

### 阮如月 · 庙祝

> 游走于庙宇与渡口间的年轻庙祝，以五情印记渡化困于世间的魂魄。

- **HP** 80 · **能量** 3 · **起手** 5张
- **核心机制**：五情印记系统 — 对敌人施加「悲/惧/怒/喜/定」印记，积累至阈值触发共鸣，造成对应效果
- **情绪亲和**：悲、惧、喜、定
- **胜利偏向**：渡化路线

**变体牌组：光愈流**（15张）
以喜印+定印为核心，通过「喜祷→花开→喜溢→光与愈」循环持续回血和触发渡化。

**专属遗物**（12件）：

| 稀有度 | 遗物 |
|--------|------|
| 普通 | 旧铜铃、观音泥、碎镜片、破庙门槛 |
| 稀有 | 问香炉、走魂册（残页）、五彩香灰、印堂朱砂 |
| 珍稀 | 庙祝法衣（残片）、残香（七束）、神像碎块 |
| 传说 | **观尘之镜**（所有印记共鸣阈值降至2层；五情共鸣+80%渡化进度；代价：镇压路线不可用） |

---

### 沈铁钧 · 捕快

> 已退休的老捕快，用铁链与怒气镇压不肯离去的亡魂。

- **HP** 100 · **能量** 3 · **起手** 5张
- **核心机制**：锁链系统 — 对敌人施加锁链（最高5层），每层贡献攻击溅射比例8%；怒爆（怒≥4时触发，攻击×2+溅射）
- **情绪亲和**：怒、定
- **胜利偏向**：镇压路线

**变体牌组：铁壁流**（15张）
以铁甲（固定减伤）为核心，积累铁甲后用「定而无极」转化为爆发攻击力。

**专属遗物**（12件）：

| 稀有度 | 遗物 |
|--------|------|
| 普通 | 旧捕快腰牌、铁链一段、怒气壶 |
| 稀有 | 县志一卷、铁算盘、旧案牍（未结案）、府衙封条 |
| 珍稀 | 铁手套（左）、千斤锁、定海神针（残）、怒爆计数器 |
| 传说 | **天网令**（锁链上限8层，溅射12%/层；代价：渡化路线不可用） |

### 无名 · 无面人

> 把自己情绪全部给出去的守护者，脸的位置是平的。

- **HP** 60 · **能量** 4 · **起手** 4张 · **手牌上限** 7张
- **核心机制**：「空」系统 — 独立于五情的空度变量（0–10），管理分段效果与空鸣触发
- **情绪亲和**：无（游离于五情体系外，但能操控所有情绪）
- **胜利偏向**：均衡（渡化/镇压效率相同）

#### 空度分段效果

| 空度 | 被动效果 |
|------|---------|
| 0–2 | 所有牌效果+10% |
| 3–5 | 无加成无惩罚 |
| 6–8 | 每张牌费用-1 |
| 9–10 | 所有牌费用为0；但每回合开始HP-5 |

#### 空鸣
同一回合内空度经历 0→10 全部阶段时触发：
1. 所有敌人最高情绪触发失调
2. 场上所有印记一次性触发共鸣
3. 渡化进度+25% **或** 对所有敌人25点穿甲伤害（玩家选择）
4. 空度归零，HP回复 = 归零前空度×2

**完整牌组**（28张）：空度操控×8、情绪转移×7、攻击×5、防御×4、工具×4

## 核心系统

### 五情系统（EmotionManager）

五情：**怒** / **惧** / **悲** / **喜** / **定**

- 情绪值范围：0–5，不可溢出
- 情绪值≥4触发**失调惩罚**：
  - 怒≥4：无法打出防御/定类牌
  - 惧≥4：回合开始随机弃1张牌
  - 悲≥4：无法使用任何回复牌
  - 喜≥4：敌人伤害全场×1.2
  - 定：无失调，无情绪增幅
- **主导情绪增幅**：怒→攻击×1.3、惧→摸牌+2、悲→吸取×1.5、喜→回复×1.5、定≥3→全牌降费

### 印记系统（阮如月专属）

- 对敌人施加「悲/惧/怒/喜/定」印记，每种印记独立计层
- 达到共鸣阈值（默认3层，遗物可降低）触发对应效果：
  - 悲共鸣：造成8伤
  - 惧共鸣：跳过敌人下次行动
  - 怒共鸣：15点穿甲伤害
  - 喜共鸣：回复8HP
  - 定共鸣：本回合+1费
- **五情共鸣**：五种印记同时在场 → 渡化进度+50%（或+30%，取决于遗物）

### 锁链系统（沈铁钧专属）

- 锁链层数上限：5层（天网令传说遗物：8层）
- 溅射比例：8%/层（天网令：12%/层）
- **怒爆**（怒≥4）：当回合攻击×2 + 溅射比例×1.5

### 铁甲系统（铁壁流新增）

- 铁甲：固定减伤值，独立于护盾
- 通过「铁甲覆体」将护盾40%转化为铁甲
- 「定而无极」可将铁甲总量转化为本回合攻击力加成（铁甲归零）

|------|------|
| 印记引爆 | 阮如月施印 → 沈铁钧锁链伤害+15% |

| 协同五情共鸣 | 两人情绪合计≥10 → 渡化进度+40%+溅射+30% |
| 协同专属遗物 | 渡魂二人券 / 铁印同心 / 并肩渡魂（传说） |

## 代码架构

```
scripts/
├── autoload/              # 全局单例
│   ├── EmotionManager.gd  # 五情状态机

│   ├── CardDatabase.gd    # 牌卡数据库（从JSON加载）
│   ├── GameState.gd       # 游戏状态/存档
│   ├── RelicManager.gd    # 遗物效果（34件）
│   ├── BuffManager.gd     # Buff/Debuff管理
│   ├── WumianManager.gd   # 无面人·空系统
│   ├── CoopManager.gd     # 协作模式管理
│   ├── AchievementManager.gd
│   ├── SoundManager.gd
│   └── TransitionManager.gd
│
├── battle/                # 战斗系统
│   ├── BattleScene.gd         # 战斗主场景
│   ├── BattleStateMachine.gd  # 战斗状态机（单人）
│   └── CoopBattleStateMachine.gd  # 战斗状态机（协作）
│
├── ui/                    # UI 组件
│   ├── BossDialogueUI.gd  # Boss 对话系统
│   ├── EmotionRadar.gd    # 五情雷达图
│   ├── PlayerHealthBar.gd / EnemyHealthBar.gd
│   ├── CardUI.gd / CardUINode.gd
│   └── UIConstants.gd     # 全局色板/字号/样式工厂
│
├── map/MapScene.gd        # 地图导航
├── event/EventScene.gd    # 民俗事件
├── shop/ShopScene.gd      # 商店
├── rest/RestScene.gd      # 休息
└── ui/EndingScene.gd      # 结局

data/
├── cards.json             # 共享+专属卡牌（80张）
├── wumian_cards.json      # 无面人专属卡牌（28张）
├── relics.json            # 遗物数据（34件）
├── coop_relics.json       # 协作遗物（3件）
├── enemies.json           # 敌人数据（11个含3 Boss）
├── boss_dialogues.json    # Boss 渡化/镇压对话文本
├── events.json            # 随机事件
└── characters.json        # 三角色基础数据
```

## 运行方法

1. 安装 [Godot 4.6](https://godotengine.org/)
2. 克隆仓库：
   ```bash
   git clone https://github.com/qianhua76123-pixel/soul-ferry.git
   ```
3. 用 Godot 编辑器打开项目（`project.godot` 在根目录）
4. 运行主场景（F5 或点击播放按钮）

> ⚠️ 项目使用 GDScript 4 严格模式，需要 Godot **4.6** 及以上版本。

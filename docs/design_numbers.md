## 3. 五情情绪系统

### 基本规则

- 五种情绪：**怒（rage）/ 惧（fear）/ 悲（grief）/ 喜（joy）/ 定（calm）**
- 每种情绪值范围：**0 ~ 5**
- 主导情绪：当前四情（除定）中值最高者。若同值且旧主导在其中，不切换。
- **定情绪**：不参与主导判定，独立作用。

### 每回合自然衰减

回合结束时，**非主导、非定**的情绪各 **-1**（最低 0）。主导情绪不衰减。

若玩家持有「执念」Buff，跳过本回合情绪衰减。

### 主导情绪加成

| 主导情绪 | 效果 |
|---|---|
| 怒（rage） | 攻击伤害 × **1.3** |
| 喜（joy） | 回复 HP × **1.5** |
| 悲（grief） | 吸血量 × **1.5**（attack_lifesteal 效果） |
| 惧（fear） | 每回合额外摸牌 **+2** |

### 定情绪加成

| 条件 | 效果 |
|---|---|
| 定 ≥ 3 | 每回合费用上限 **-1**（即费用从3降为2） |

### 失调机制（Disorder）

情绪值 ≥ **4** 时进入失调状态：

| 失调情绪 | 限制 |
|---|---|
| 怒失调 | 无法打出**定系牌**，无法打出**护盾牌** |
| 悲失调 | 无法打出**治疗牌**（heal/heal_all_buffs） |
| 惧失调 | 每回合强制弃置**1张随机手牌** |
| 喜失调 | 敌人造成伤害 × **1.2** |
| 定失调 | 无效果（代码注释：定不参与失调） |

## 4. 卡牌系统

### 稀有度

| 稀有度 | 说明 |
|---|---|
| common（普通） | 基础牌，数量最多 |
| rare（稀有） | 中端牌，有条件加成或特殊效果 |
| legendary（传说） | 强力牌，费用较高 |
| epic（史诗） | 角色专属高级机制牌 |
| story（剧情） | 仅通过事件获得 |

### 升级效果类型

| 类型 | 效果 |
|---|---|
| power | 数值提升（+25% 或固定加值） |
| cost | 费用 -1 |
| extend | 追加额外效果（如额外抽牌/护盾持续等） |
| unlock | 移除条件限制（原有条件变为无条件） |
| transform | 彻底重铸，效果全面提升 |

### 卡牌条件触发

| 条件 | 触发 |
|---|---|
| `rage_dominant` | 怒为主导情绪 |
| `fear_dominant` | 惧为主导情绪 |
| `grief_dominant` | 悲为主导情绪 |
| `joy_dominant` | 喜为主导情绪 |
| `calm_dominant` | 定为主导情绪 |
| `rage >= 3` | 怒 ≥ 3 |
| `calm >= 3` | 定 ≥ 3 |
| `grief >= 3` | 悲 ≥ 3 |
| `grief >= 2` | 悲 ≥ 2 |
| `fear >= 3` | 惧 ≥ 3 |
| `fear >= 2` | 惧 ≥ 2 |
| `joy >= 3` | 喜 ≥ 3 |

| 分段 | 范围 | 效果 |

### 空鸣触发条件

**同一回合内**，空度经历：最小值 ≤ 2 且 最大值 ≥ 9

（即同一回合内空度从低（≤2）被推至高（≥9），或反之）

### 空鸣效果

1. 回复 HP = **空鸣前空度 × 2**
2. 空度归零
3. 激活「镜·无我」持续 **3 回合**（50%概率受伤减半）
4. 让玩家选择：**渡化进度 +25%** 或 **造成25点穿甲伤害**

### 受伤修正

当空度 ≥ 6（高段）时：受到伤害 **-20%**

每次受到伤害：空度 **-1**

### 镜·无我（Kongming Mirror）

激活后持续3回合，每次受伤时50%概率**伤害减半**。

## 13. 渡化与镇压判定

### 渡化（Du Hua）

渡化是胜利路线之一，不伤害而是"感化"敌人，结局更正面。

| 条件类型 | 触发方式 |
|---|---|
| `emotion_threshold` | 情绪值满足要求时自动触发 |
| `card_play` | 情绪满足时打出特定牌 |
| `consecutive_joy_cards` | 情绪满足时连续打出N张喜系牌 |
| `dialogue_choice` | 情绪满足时选择特定对话选项 |
| `multi_stage` | 分3阶段递进满足情绪条件 |

渡化成功奖励（`渡魂册`在背包时）：**永久 +3 最大HP**

### 镇压（Zhen Ya）

将敌人HP降至0，传统战斗胜利方式。

### 路线倾向计算

```
du_hua_count vs zhen_ya_count

若 du_hua > zhen_ya × 2 → route_tendency = "du_hua"
若 zhen_ya > du_hua × 2 → route_tendency = "zhen_ya"
其他 → route_tendency = "mixed"
```
